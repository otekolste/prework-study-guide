HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The head element contains meta elements, which store metadata that is not visible to the user.
  • The body element is an example of a "semantic" element, wherein the name of the element describes its content and the role it plays on the webpage. It is important to use semantic elements when possible.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: tracks all files (specific file names can be put in to track only those files)
  • git commit -m "comment": commits files, with description as defined in comment
  • Sync your local code with the remote version often!
  • Make sure to make changes in branches, NOT main, so if something goes wrong it's easy to revert and doesn't break everythiing.
